# Amneziawg

## Native integrations

Gluetun supports Amneziawg with the custom provider only for now.

Note, depending on how it's configured, it can be used with a Wireguard server.

See [the custom provider](providers/custom.md) for more details on how to use this.

## Available options

Available options are listed in [options/amneziawg](options/amneziawg.md).
